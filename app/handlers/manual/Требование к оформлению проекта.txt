<p><b>ТРЕБОВАНИЯ К ОФОРМЛЕНИЮ ПРОЕКТА</b><br />Перед защитой проектов капитанам нужно будет сделать и загрузить пояснительную записку<br />Так же для защиты проекта командам нужна презентация по проделанной работе<br /><b>1. Пояснительная записка по проекту должна быть оформлена с соблюдением требований и содержать следующие разделы:</b></p>
<ul>
<li>титульный лист;</li>
<li>список исполнителей с указанием обязанностей по реализации проекта;</li>
<li>реферат;</li>
<li>содержание;</li>
<li>определения, обозначения и сокращения (вариативно);</li>
<li>техническое задание на проект;</li>
<li>введение;</li>
<li>основная часть;</li>
<li>результаты проделанной работы (заключение);</li>
<li>список использованных источников (вариативно);</li>
<li>приложения.</li>
</ul>
<p><b>2. Структура презентации проекта должна включать разделы:</b></p>
<ul>
<li>команда проекта;</li>
<li>техническое задание;</li>
<li>проблема;</li>
<li>предлагаемая технология решения проблемы;</li>
<li>продукт проекта;</li>
<li>оценка рынка;</li>
<li>план коммерциализации (для второго творческого проекта, если проект предусматривает коммерциализацию);</li>
<li>свободный раздел;</li>
<li>команда проекта / заключение.</li>
</ul>
